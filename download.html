Download

The project provides custom Godot builds with Goost already pre-compiled. Currently, only stable versions of the engine are released. If you'd like to compile Godot with Goost from source yourself, proceed to the official Goost documentation to learn how to do this.

If you have any doubts regarding what you need to download, you can refer to Godot's download page as well.

Warning: Goost is still considered in early development phase. While the Goost binaries were built using the official Godot build containers, not all of them are properly tested, and they are currently not code-signed.
